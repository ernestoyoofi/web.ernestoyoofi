## Apakah google drive bisa menjadi cloud media api kita?

Tentu saja bisa!, Google menyediakan Google Drive API yang memungkinkan pengembang mengakses dan mengelola file di Google Drive melalui aplikasi atau layanan yang mereka bangun.

Dengan menggunakan Google Drive API, kita dapat membuat aplikasi yang berinteraksi dengan file media di Google Drive, seperti mengupload file dan melihat file, ataupun mengolahnya. Namun yang kita akan gunakan hanyalah baca file (bytes file) & menulis (upload file).
